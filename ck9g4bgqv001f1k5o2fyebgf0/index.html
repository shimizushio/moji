有妳的世界

发表于 2018-06-06 更新于 2020-04-26 分类于 同人 ， 莲家

Lofter链接：http://shiorc.lofter.com/post/1f0e7808_ee7fea93

——【以下正文】——

「戴森姊姊，起床了！」
被從美夢中吵醒的戴森迷迷糊糊地聽著耳邊喊著起床的聲音，睜開了眼睛。
眼前的模模糊糊的身影逐漸變得清晰起來，克拉克斯頓姊姊正趴在她的身上，用手裡拿著的蠟筆戳了戳戴森柔軟的臉蛋。
隱約感覺到哪裡不對勁的戴森對著面前的小女孩發了一會兒呆，突然意識到她的身體變小了的事情。
揉了揉眼睛的戴森確認了眼前的這個小傢伙真的就是自己親愛的克拉克斯頓姊姊之後，嘴裡吐出一句「那麼這就還是在夢裡了」之後又躺了回去閉上眼。
「喂，戴森！起床了！」
————
變小藥這種東西戴森是有印象的，大約半年前偷偷在蓮大人的飯裡混進變小藥之後發生的事情她也都清晰地記得。可那只是讓身體縮小的藥，而不是像現在這樣。
戴森難得地在臉上掛滿煩惱的陰雲，這一切的起因都是這個她親愛的姊姊身上發生的異狀。
——她也不知道現在是叫她姊姊好還是妹妹好了。
克拉克斯頓的身體不是縮小，而是變回了小時候的樣子，連帶著精神上也似乎變成了小孩子的樣子。剛剛自己躺回去睡了之後，克拉克斯頓便氣鼓鼓地在她臉上畫滿了塗鴉，她不得不在徹底清醒以後花上不少時間先把那些塗鴉給清洗乾淨。
戴森看著小克拉克斯頓踮腳試圖去夠那個她夠不到的掛在牆上的廚具的樣子，又想起平日裡那個什麼家務都輕鬆地一手包辦的克拉克斯頓，這兩個相似卻不完全一樣的身影在她的眼前重合，她的思緒不禁飛得稍微遠了一些。
如果是這個小克拉克斯頓的話，說不定戴森說什麼她都會聽呢。
戴森悄悄地從身後靠近了還在拼命踮腳的小克拉克斯頓，把她從墊腳的凳子上抱了下來。
「戴森姊姊，做什麼啦，放開我！」
「聽話哦，小克拉克。」戴森的語氣是她自己都未曾聽過的溫柔，卻意外地與一個半月之前收到的那件克拉克斯頓姊姊作為生日禮物給她挑選的、現在正穿在她身上的雪白長裙非常合拍。在生日當天的時候蓮大人一臉興奮地用畫筆將她的樣子記錄下來還讓她有點害羞，不過現在在小克拉克斯頓的面前，那份自己的氣質會不會和這衣服不搭的擔心已經被拋到了腦後。
「做飯的事情就交給我吧，小克拉克就乖乖在旁邊等著。」
戴森的這句話讓被她攔腰抱起還在揮著小拳頭抗議的小克拉克安靜了下來。她水汪汪的眼睛和戴森對視了一下，怯生生地點了點頭，退到了一旁。
戴森一邊穿上克拉克斯頓平時在廚房用的圍裙，一邊在心底裡暗自開心。小克拉克是真的不會反抗她說的話，就連做飯這種平日的她絕不會簡單地讓出來的工作也…這麼說的話…
腦內的金髮少女蹲下來親了親小女孩臉頰的場景讓戴森的臉頰有些發燙，她搖了搖頭把這個想法從腦海裡趕了出去，被這個動作帶起的金髮也隨之像海浪一樣波動。真是奇怪，明明向克拉克斯頓姊姊告白的時候能面不改色地說出那樣的話，現在害羞的反倒是戴森自己了。
儘管在料理上戴森明白她要學的還有很多，但總歸今天是超常發揮了一下，讓小克拉克在盤子空了以後露出了滿足的微笑。
吃過早飯，清洗好碗碟，脫下圍裙的戴森回到客廳，看見的是小克拉克趴在地上用蠟筆畫繪本的場景。
家事萬能的本領受到身體的變化發揮不出來，可那高超的繪畫技巧卻好像深深刻進了克拉克斯頓的靈魂，就算精神也變成了小克拉克，她長年積累的繪畫經驗可沒有一併失去。戴森看著小克拉克繪畫的場景，不禁走到她的身後，輕輕靠在了小傢伙的身上。
不管對小克拉克做什麼都可以，不會被拒絕的…或許是這樣的吧。
她做好了覺悟，在小克拉克的耳邊輕聲呢喃。
「小克拉克，我可以和妳一起畫畫嗎？」
「好呀！」
這正是她所期待的回答。
她的稍微大一點的手，覆在小克拉克小小的手上，一同拿住紅色的蠟筆。
用幸福的顏色作畫，在繪本上留下了只有她們兩人的世界。
戴森感受著掌心傳來的小克拉克小手的溫度，沈醉於其中久久不願放開她的手。
————
愉快的兒童節結束的時候，在戴森懷裡的小克拉克抬頭望了望雙臂環繞著她已經沈沈入睡的戴森。她的身體還是那副小小的樣子，可此刻的目光如果被戴森看到就會明白，藏在這幅身體裡的靈魂絲毫沒有受到影響。
她可是克拉克斯頓，只要她想，她就能把在她掌心的戴森騙得團團轉。她悄悄伸手，撫摸著在一個半月前的生日當天，她往戴森的左手無名指上套上的小金屬環。
戴森覺得今天的小克拉克會對她言聽計從，可這一切都是克拉克斯頓算計好的，從她私下聯繫夕張博士弄到手的新型的變小藥開始，到那本繪本上今天新留下的痕跡為止。
一切都是克拉克斯頓所營造好的謊言，沒有什麼一覺醒來突然發覺身體變小了的情節，全都是編織好的故事。
不過，這個謊言的目的也絕不是什麼難以啟齒的陰暗內容。克拉克斯頓只是希望，與戴森在一起的每一天，都是那麼地有趣，永遠不會感到厭倦。這樣的話，她掌心的溫度就是永遠只屬於我的，只屬於克拉克斯頓的東西。

# Dyson # Claxton
